# Scripts Directory

This directory contains additional utility scripts for the LipSyncAutomation project.

## Current Scripts

### Complex/Backup Scripts
- `start_web_ui_complex.sh` - The original complex startup script (backup)
- `setup_dependencies.sh` - Comprehensive dependency installation
- `run_tests.sh` - Advanced test runner
- `cleanup.sh` - System cleanup utility

## Primary Scripts

The main scripts are now in the project root:
- `../setup_web_ui.sh` - One-time setup (recommended)
- `../start_web_ui.sh` - Daily startup (recommended)
